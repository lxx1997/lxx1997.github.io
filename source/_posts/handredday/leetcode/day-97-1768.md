## 题目名称

{% blockquote %}
给你两个字符串 word1 和 word2 。请你从 word1 开始，通过交替添加字母来合并字符串。如果一个字符串比另一个字符串长，就将多出来的字母追加到合并后字符串的末尾。

返回 合并后的字符串 。
{% endblockquote %}

## 示例

{% blockquote %}
输入：word1 = "abc", word2 = "pqr"
输出："apbqcr"
解释：字符串合并情况如下所示：
word1：  a   b   c
word2：    p   q   r
合并后：  a p b q c r
{% endblockquote %}


## 题解


{% blockquote %}

略

{% endblockquote %}

## 答案

~~~js
/**
 * @param {string} word1
 * @param {string} word2
 * @return {string}
 */
var mergeAlternately = function(word1, word2) {
  let word = ""
  let length = Math.min(word1.length, word2.length)
  let i = 0
  for(; i < length; i++) {
    word += word1[i] + word2[i]
  }
  if(i < word1.length) {
    word += word1.slice(i)
  }
  if(i < word2.length) {
    word += word2.slice(i)
  }
  return word
};
~~~
