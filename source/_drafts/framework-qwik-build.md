Qwik 是一種新型 Web 框架，可以提供任何大小或複雜性的即時加載 Web 應用程序。您的網站和應用程序可以使用大約 1kb 的 JS（無論應用程序複雜性如何）啟動，並在規模上實現一致的性能。
